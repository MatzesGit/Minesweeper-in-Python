This is simple example how to use Python with Tkinter add on. 

https://github.com/MatzesGit/Minesweeper-in-Python.git


To use this you need to install Python version 3+

Start the program with the minesweeper.py

all imports will be done inside the Plugin folder - plugin.py please check the ability for the Python libaries

***** Game controls: *****

With the "left mouse button" you can open the hidden fields

With the "right mouse button" you can mark the possible bomb.

After marking all bombs correctly, be happy you win the game
